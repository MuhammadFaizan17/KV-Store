🔑 Secretlab Key-Value Store API

Status: ✓ Running

A production-quality, version-controlled key-value store with time-travel query capabilities.

📚 API Endpoints

POST /api/v1/object

Store key-value pairs with UTC timestamp

GET /api/v1/object/{key}

Get the latest value for a key

GET /api/v1/object/{key}?timestamp={unix}

Get value at a specific point in time (time-travel query)

GET /api/v1/object/get_all_records

Get the latest value for all keys

🚀 Quick Start

# Store a value
curl -X POST http://localhost:8000/api/v1/object \
  -H "Content-Type: application/json" \
  -d '{"mykey": "myvalue"}'

# Get the latest value
curl http://localhost:8000/api/v1/object/mykey

# Get all records
curl http://localhost:8000/api/v1/object/get_all_records
